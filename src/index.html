Laptop Rental is a PHP Laravel web application following the MVC architecture with a MySQL database. It enables university campus students to safely share laptops, making resources accessible to everyone. GambleAware

Gamble Aware is a Python-based data science project developed using matplotlib, scikitlearn, decisiontreeclassifier and Streamlit libraries. Following the CRISP-DM methodology and SCRUM for project management, the project focuses on diagnosing problematic online gamblers. The system aims to bring accountability to the online gambling industry by providing early detection and intervention for individuals exhibiting problematic gambling behaviors.
